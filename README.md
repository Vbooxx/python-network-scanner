# python-network-scanner
This project is a simple Python-based network scanner that identifies active devices on a local network.   The program scans a user-specified IP range and sends ping requests to determine which hosts are online.

# Python Network Scanner

This project is a simple Python network scanner that identifies active devices on a local network.

## Features
- Scans an IP range
- Uses ping to detect active hosts
- Displays live devices on the network

## Technologies
- Python
- ipaddress module
- subprocess module

## Example

Run the scanner:

python network_scanner.py

Enter network range:

192.168.1.0/24

The scanner will check each IP address and report active devices.
